# E-commerce Application

This repository contains a Spring Boot e-commerce project with a refreshed UI and improved media handling.

## Recent Updates
- Modernized UI pages (shop, product view, cart, login, registration).
- Added checkout page and consistent styling.
- Fixed rupee symbol rendering and UI icons.
- Improved image handling with support for URL-based images and safe fallbacks.

## Run
Use Spring Tool Suite or run:

```bash
./mvnw spring-boot:run
```
